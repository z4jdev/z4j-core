# z4j-core

[![PyPI version](https://img.shields.io/pypi/v/z4j-core.svg)](https://pypi.org/project/z4j-core/)
[![Python](https://img.shields.io/pypi/pyversions/z4j-core.svg)](https://pypi.org/project/z4j-core/)
[![License](https://img.shields.io/pypi/l/z4j-core.svg)](https://github.com/z4jdev/z4j-core/blob/main/LICENSE)

The z4j domain core — shared models, protocols, transport, redaction, and policy.

Pure-Python dependency for every other z4j package. Has no
framework / engine / database imports, so it can be vendored
into agent processes without dragging server runtime weight.
End users do not install this package directly; it comes in
through the brain or any agent package.

## Install

```bash
pip install z4j-core
```
